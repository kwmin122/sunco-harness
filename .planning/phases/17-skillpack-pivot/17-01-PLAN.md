All CJS format (.cjs), standalone, no external deps.

1. sunco-check-update.cjs (SessionStart):
   - Read VERSION from sunco/VERSION
   - npm view popcoru version (4s timeout)
   - Output update notice if newer version exists
   - Exit silently on any error

2. sunco-statusline.cjs:
   - Read .planning/STATE.md from cwd
   - Output: "SUNCO | Phase N: Name | XX%"
   - Fallback: "SUNCO | No project"

3. sunco-context-monitor.cjs (PostToolUse):
   - Track context window usage from tool results
   - Warn when approaching limits (70%, 85%, 95%)
   - Suggest /sunco:pause-work when critical

4. sunco-prompt-guard.cjs (PreToolUse on Write/Edit):
   - Scan content being written to .planning/ for injection patterns
   - Advisory warning (not blocking)
   - Patterns: system prompt overrides, role reassignment, ignore instructions
  </action>
