## Release Truth

The v1.6/M8 release artifact is `popcoru@0.14.0`.

Release truth is split deliberately:

- **Source tree**: build/typecheck/test/lint/audit/artifact gates must pass.
- **Packed artifact**: `npm run test:artifact --workspace popcoru` must pass from a clean npm prefix and temp HOME.
- **Registry artifact**: after `v0.14.0` is published, `npm run test:artifact --workspace popcoru -- --registry popcoru@0.14.0` must pass.
- **External authority**: npm publish requires either local npm authentication or a GitHub `NPM_TOKEN` secret.

On this machine, `npm whoami` currently returns `E401` and `gh secret list --repo kwmin122/sunco-harness` shows no configured `NPM_TOKEN`, so the external publish step is blocked until npm credentials are provided. The release gate and tag workflow are prepared so publish can proceed immediately once that credential exists.
