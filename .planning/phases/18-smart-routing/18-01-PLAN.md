### Task 1: Intent Classifier
**read_first**: packages/cli/hooks/sunco-mode-router.cjs
**files**: packages/skills-workflow/src/shared/intent-classifier.ts (NEW)

Deterministic intent classification (zero LLM cost):
```typescript
export type IntentType = 'lookup' | 'implement' | 'investigate' | 'plan' | 'review'
export interface IntentResult {
  intent: IntentType
  confidence: number  // 0-1
  keywords: string[]  // matched keywords
}
export function classifyIntent(input: string): IntentResult
```

Keyword patterns per intent:
- lookup: find, where, show, list, what is, 어디, 찾아, 보여줘
- implement: add, create, write, build, fix, make, 만들어, 추가, 수정
- investigate: debug, why, error, fail, broken, 왜, 에러, 버그
- plan: plan, design, architect, roadmap, 계획, 설계
- review: review, check, verify, audit, 검토, 확인

**acceptance_criteria**:
- GREPPABLE: `classifyIntent` exported
- Unit tests for all 5 intents + Korean keywords
