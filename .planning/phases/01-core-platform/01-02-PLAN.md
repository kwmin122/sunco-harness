  <behavior>
    **deepMerge:**
    - deepMerge({a: 1}, {b: 2}) => {a: 1, b: 2}
    - deepMerge({a: {x: 1}}, {a: {y: 2}}) => {a: {x: 1, y: 2}} (deep merge objects)
    - deepMerge({a: [1, 2]}, {a: [3, 4]}) => {a: [3, 4]} (array REPLACE, per D/CFG-02)
    - deepMerge({a: {x: 1}}, {a: [1]}) => {a: [1]} (override type wins)
    - deepMerge({}, {a: 1}) => {a: 1}
    - deepMerge({a: 1}, {}) => {a: 1}

    **loadConfig:**
    - loadConfig with no TOML files => returns SunConfigSchema defaults (all fields populated)
    - loadConfig with only global config => global values override defaults
    - loadConfig with all 3 layers => directory overrides project overrides global
    - loadConfig with invalid TOML syntax => throws ConfigError with file path and line number
    - loadConfig returns Object.freeze() result (immutable)

    **schema validation:**
    - Valid partial config => fills missing fields with Zod defaults
    - Invalid type (agent.timeout = "slow") => throws with user-friendly message containing field path
    - Empty object {} => returns full default config
  </behavior>
  <implementation>
    **merger.ts:**
    - Export function deepMerge(base, override) implementing object-deep-merge + array-replace
    - ~30 lines. Recursive. Arrays and non-object values replace. Objects merge.

    **schema.ts:**
    - Re-export SunConfigSchema from config/types.ts (or define full schema here)
    - Export validateConfig(raw: unknown): SunConfig that calls SunConfigSchema.parse()
    - On ZodError, catch and re-throw as ConfigError with z.prettifyError() message

    **loader.ts:**
    - Export async function loadConfig(cwd: string): Promise<Readonly<SunConfig>>
    - Load 3 layers using smol-toml parse(): os.homedir()/.sun/config.toml, findProjectRoot(cwd)/.sun/config.toml, cwd/.sun.toml
    - findProjectRoot: walk up from cwd looking for .sun/ directory or package.json
    - Skip missing files silently (catch ENOENT)
    - On TOML parse error: throw ConfigError with file path + smol-toml TomlError line/column
    - Merge: deepMerge(deepMerge(global, project), directory)
    - Validate: validateConfig(merged)
    - Freeze: Object.freeze(result) before return

    **index.ts:**
    - Re-export loadConfig, deepMerge, validateConfig, SunConfigSchema, SunConfig
  </implementation>
</feature>
