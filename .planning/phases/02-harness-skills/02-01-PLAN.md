    **Ecosystem Detector** (packages/skills-harness/src/init/ecosystem-detector.ts):
    - Export `async function detectEcosystems(opts: { cwd: string }): Promise<EcosystemResult>`
    - Iterate ECOSYSTEM_MARKERS, check each with `fs.access(path.join(cwd, marker.file))`. For glob patterns like `*.csproj`, use `glob` to check existence.
    - Return deduplicated ecosystem names, all matching markers, and primaryEcosystem (first high-confidence match).
    - Per D-01: 15+ ecosystem markers. The types.ts ECOSYSTEM_MARKERS constant provides the data.

    **Layer Detector** (packages/skills-harness/src/init/layer-detector.ts):
    - Export `async function detectLayers(opts: { cwd: string }): Promise<LayerResult>`
    - Scan for common source roots: 'src', 'lib', 'app', 'packages'. Pick first that exists as sourceRoot.
    - Within sourceRoot, read top-level directories with `fs.readdir`.
    - For each directory, match case-insensitively against COMMON_LAYER_PATTERNS[].dirPatterns.
    - For each match, create a DetectedLayer with: name from COMMON_LAYER_PATTERNS, pattern = `{sourceRoot}/{actualDirName}/*`, canImportFrom from COMMON_LAYER_PATTERNS.
    - Per D-02: directory name heuristics. Import graph sampling is secondary -- skip for v1 if directory heuristics suffice.

    **Convention Extractor** (packages/skills-harness/src/init/convention-extractor.ts):
    - Export `async function extractConventions(opts: { cwd: string }): Promise<ConventionResult>`
    - Per D-03: AST-free regex analysis. Sample up to 50 .ts/.tsx/.js/.jsx files from src/ (exclude node_modules, dist, build, .next, vendor, __fixtures__, __mocks__).
    - **Naming**: regex `/^export (?:function|const|class|type|interface) ([a-zA-Z_$][a-zA-Z0-9_$]*)/gm` on file contents. Classify each name: camelCase if /^[a-z]/ + has uppercase, PascalCase if /^[A-Z]/, snake_case if /_/ and all lowercase, kebab-case if /-/. Vote by majority.
    - **Import style**: regex `/^import .+ from ['"](.+)['"]/gm`. If path starts with './' or '../' = relative, else alias. Vote.
    - **Export style**: count `export default` vs `export { }` / `export function/const`. Vote.
    - **Test organization**: check for `__tests__/` directories = '__tests__', `*.test.ts` alongside source = 'co-located', top-level `test/` or `tests/` = 'top-level-test', else 'unknown'.

    **Tests**: Use Vitest with temporary directories created via `fs.mkdtemp`. Create minimal fixture files (package.json, tsconfig.json, src/ with subdirectories) to exercise each detector. Clean up in afterEach.
