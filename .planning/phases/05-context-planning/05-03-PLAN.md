    Step 0: Entry + provider check

    Step 1: Determine phase number (same pattern as discuss/assume)

    Step 3: Determine research topics (D-11).
    - If `ctx.args.topics` provided: split by comma, trim, use those topics

    Step 4: Parallel research dispatch (D-08, D-09). For each topic:
    ```typescript
    const RESEARCH_PERMISSIONS: PermissionSet = {

      allowTests: false,
      allowNetwork: false,
      allowGitWrite: false,
      allowCommands: [],
    };

    const results = await Promise.allSettled(

        const result = await ctx.agent.run({
          role: 'research',
    ```
