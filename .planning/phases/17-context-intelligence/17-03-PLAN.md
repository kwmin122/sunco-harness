### Task 4: Integrate Context Zones into Auto Skill
**read_first**: packages/skills-workflow/src/auto.skill.ts, packages/skills-workflow/src/shared/context-zones.ts
**files**: packages/skills-workflow/src/auto.skill.ts

Add context zone checking to auto pipeline:
1. Before each step dispatch: read `.sun/context-zone.json`
2. At Orange zone (70-85%): log warning, save HANDOFF.json automatically
3. At Red zone (85%+): auto-pause pipeline, write HANDOFF with resumeCommand
4. Pass contextZone to skills via args for selective loading decisions

Add to PIPELINE_STEPS dispatch:
```typescript
const zoneFile = await readContextZone(ctx)
if (zoneFile?.zone === 'red') {
  // Auto-save and suggest resume
  await ctx.run('workflow.pause')
  break // Exit pipeline, user resumes in new session
}
```

**acceptance_criteria**:
- GREPPABLE: `context-zone.json` read in auto.skill.ts
- Red zone triggers auto-pause in auto pipeline
- Orange zone triggers warning + HANDOFF save
