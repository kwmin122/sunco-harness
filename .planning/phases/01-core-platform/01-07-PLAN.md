    Per D-32, implement the 4 core patterns that map to the skill state machine:
    `idle -> entry -> choice? -> running -> result`

    **packages/core/src/ui/patterns/SkillEntry.tsx:**
    Renders a brief entry banner when a skill starts. Shows skill title, optional description.
    - Props: SkillEntryInput (title, description?, skillId?)
    - onComplete callback when rendering finishes
    - Layout: Box with padding, primary-colored title text, dim description text
    - Auto-completes after brief display (no user interaction needed)

    **packages/core/src/ui/patterns/InteractiveChoice.tsx** (per D-36, UX-01):
    Multi-option selection with keyboard navigation and Recommended badge.
    - Props: AskInput (message, options, defaultId?)
    - onSelect callback returning UiChoiceResult
    - Use ink-select-input for the selection list
    - Each option shows: label + description + Badge("Recommended") if option.recommended=true
    - Default selection highlights defaultId option
    - Returns { selectedId, selectedLabel, source: 'keyboard' } on selection

    **packages/core/src/ui/patterns/SkillProgress.tsx** (per D-35, UX-03):
    Progress display supporting both determinate (progress bar) and indeterminate (spinner) modes.
    - Props: ProgressInput (title, total?) + current state (completed?, message?)
    - If total is provided: render progress bar showing completed/total with percentage
    - If total is undefined: render ink-spinner with message text
    - Updatable via external state changes (parent component manages state)

    **packages/core/src/ui/patterns/SkillResult.tsx** (per UX-02, UX-03):
    Final output after skill execution. Shows summary and optional recommendation cards.
    - Props: ResultInput (title, summary, details?, recommendations?)
    - Layout: StatusSymbol(success) + title, summary text, optional details in dim text
    - If recommendations provided: render RecommendationCard component below

    **packages/core/src/ui/hooks/useSelection.ts:**
    Custom hook for keyboard-driven selection in lists.
    - useSelection(items, defaultIndex) returns { selectedIndex, selectedItem, handlers }
    - Handles up/down arrow keys, enter for confirmation

    **packages/core/src/ui/hooks/useKeymap.ts:**
    Custom hook for keyboard shortcuts.
    - useKeymap(keymap: Record<string, () => void>) attaches key handlers via Ink's useInput

    **packages/core/src/ui/session/StatusBar.tsx** (per D-37):
    Session-level UI showing model usage and context. NOT part of skill lifecycle -- lives at runtime level.
    - Props: { provider?: string; tokenUsage?: number; contextUsage?: number; cost?: number }
    - Renders at bottom: provider name + token % + context % + cost
    - This is a standalone component skills do NOT import. The CLI runtime manages it.

    **packages/core/src/ui/patterns/index.ts and hooks/index.ts:** Barrel exports.
