    **Freshness Checker** (packages/skills-harness/src/health/freshness-checker.ts):
    Per D-12:
    - Export `async function checkFreshness(opts: { cwd: string }): Promise<FreshnessResult>`
    - Find documentation files: glob `**/*.md` + `**/docs/**` (exclude node_modules, dist, .sun).
    - For each doc file, use `fs.stat` to get mtime.
    - Find related code files: same directory or referenced via relative links in the doc.
    - Compare doc mtime vs code mtime. If code modified > 7 days after doc, mark as stale.
    - Check cross-references: regex scan for `[text](./path)` and `@see path` patterns, verify target exists with `fs.access`.
    - Score = 100 * (1 - staleCount / totalDocs). Broken references reduce score by 5 each.

    **Pattern Tracker** (packages/skills-harness/src/health/pattern-tracker.ts):

    - Export `async function trackPatterns(opts: { cwd: string; state: StateApi }): Promise<PatternCount[]>`
    - Scan source files (glob `src/**/*.{ts,tsx,js,jsx}`, exclude node_modules/dist).
    - Track these anti-patterns via regex:

    **Health Skill** (packages/skills-harness/src/health.skill.ts):
    ```typescript
    defineSkill({
      id: 'harness.health',
      command: 'health',
      kind: 'deterministic',
      stage: 'stable',
      category: 'harness',
      routing: 'directExec',
      description: 'Codebase health score -- freshness, anti-patterns, conventions',
      options: [
        { flags: '--json', description: 'Output health report as JSON' },
        { flags: '--no-snapshot', description: 'Skip storing snapshot (dry run)' },
      ],
      async execute(ctx) {
        // 1. Load init result for convention baseline
        const initResult = await ctx.state.get<InitResult>('init.result');
        // 2. Run all checks in parallel
        const [freshness, patterns, conventionResult] = await Promise.all([
          checkFreshness({ cwd: ctx.cwd }),
          trackPatterns({ cwd: ctx.cwd, state: ctx.state }),
          scoreConventions({ cwd: ctx.cwd, expectedConventions: initResult?.conventions ?? defaultConventions }),
        ]);
        // 3. Get trends from previous snapshots
        const trends = await getPatternTrends(ctx.state);
        // 4. Compute scores
        const patternScore = computePatternScore(patterns, previousPatterns);
        const overallScore = computeHealthScore({ freshness, patternScore, conventionScore: conventionResult.score });
        // 5. Build report
        const report: HealthReport = { overallScore, freshness: {...}, patterns: {...}, conventions: {...}, trend: ... };
        // 6. Store snapshot (unless --no-snapshot)
        if (!ctx.args.noSnapshot) {
          const snapshot: HealthSnapshot = { date: new Date().toISOString(), freshness, patterns, conventionScore: conventionResult.score, overallScore };
          await ctx.state.set(`health.snapshot.${snapshot.date}`, snapshot);
        }
        // 7. Output
        // Store for recommender
        await ctx.state.set('health.lastResult', { overallScore, timestamp: new Date().toISOString() });
        return { success: true, summary: `Health score: ${overallScore}/100`, data: report };
      },
    });
    ```
