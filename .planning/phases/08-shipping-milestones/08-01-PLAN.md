    - Export `parseGitLog(log: string): ChangelogEntry[]` -- parse conventional commit format from `git log --oneline` output. Pattern: `hash type(scope): description` or `hash type: description`. Entries that don't match pattern go under 'chore'.
    - Export `prependChangelog(existingContent: string, newSection: string): string` -- insert new section after `# Changelog` heading (or create heading if missing). Preserve existing content below.

    - Export `buildGapPhases(unmetReqs: string[], roadmapContent: string): string` -- for each unmet requirement group, call `addPhase()` from roadmap-writer to add catch-up phases. Return updated roadmap content.
