### Task 2: Cost-Aware Model Selector
**read_first**: packages/core/src/agent/router.ts, packages/skills-workflow/src/shared/budget-guard.ts
**files**: packages/skills-workflow/src/shared/model-selector.ts (NEW)

Deterministic model selection based on complexity + budget:
```typescript
export type ModelTier = 'fast' | 'balanced' | 'quality'
export function selectModelTier(opts: {
  complexity?: 'simple' | 'standard' | 'complex'
  budgetPercent: number
  intentType?: IntentType
}): ModelTier
```

Rules:
- simple + any budget → fast
- standard + budget < 75% → balanced
- standard + budget >= 75% → fast (downgrade)
- complex + budget < 75% → quality
- complex + budget >= 75% → balanced (downgrade)
- Intent override: lookup always fast, plan/review always quality

**acceptance_criteria**:
- GREPPABLE: `selectModelTier` exported
- Unit test: all complexity×budget combinations
