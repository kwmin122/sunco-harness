    **Doc Analyzer** (packages/skills-harness/src/agents/doc-analyzer.ts):
    Per D-16:
    - Export `async function analyzeAgentDoc(filePath: string): Promise<AgentDocMetrics>`
    - Read file content via `fs.readFile`.
    - Count total lines.
    - Detect sections via regex: `/^#{1,3}\s+(.+)/gm` (markdown headers).
    - For each section, count instruction lines: lines starting with `- `, `* `, numbered lists, or containing imperative verbs (`must`, `should`, `always`, `never`, `use`, `avoid`, `do not`).
    - Compute instructionDensity = total instructions / sectionCount.
    - Detect key sections: hasConventions (section title contains 'convention' or 'coding' or 'style'), hasConstraints (contains 'constraint' or 'rule' or 'requirement'), hasArchitecture (contains 'architecture' or 'structure' or 'design').
    - **Contradiction detection**: Simple heuristic -- find lines with opposing directives:
      - Build pairs from "always/never", "use/avoid", "do/don't" patterns.
      - If "always use X" appears near "never use X" or "avoid X" (within same topic/section or globally), flag as contradiction.
      - Use string similarity on the noun phrases after the verb.
    - Set lineCountWarning = totalLines > 60.

    **Efficiency Scorer** (packages/skills-harness/src/agents/efficiency-scorer.ts):
    Per D-17 and ETH Zurich insight:
    - Export `function computeEfficiencyScore(metrics: AgentDocMetrics): number`
    - Four components, each 0-100:
      1. **Brevity (30%)**: Use brevityScore function: <=30 lines=100, <=60=80, <=100=50, <=200=25, >200=10.
      2. **Clarity (25%)**: 100 - (vaguePhraseCount * 5). Vague phrases: 'properly', 'as needed', 'when appropriate', 'consider', 'ensure', 'make sure'. Each occurrence deducts 5 points (floor at 0).
      3. **Coverage (25%)**: 100 if all three key areas covered (conventions, constraints, architecture). Deduct 33 for each missing.
      4. **Contradiction-free (20%)**: 100 if no contradictions. Deduct 25 per contradiction (floor at 0).
    - Final = brevity*0.30 + clarity*0.25 + coverage*0.25 + contradictionFree*0.20. Round to integer.

    **Agents Skill** (packages/skills-harness/src/agents.skill.ts):
    ```typescript
    defineSkill({
      id: 'harness.agents',
      command: 'agents',
      kind: 'deterministic',
      stage: 'stable',
      category: 'harness',
      routing: 'directExec',
      description: 'Analyze agent instruction files (CLAUDE.md, agents.md) -- efficiency score + suggestions',
      options: [
        { flags: '--json', description: 'Output analysis as JSON' },
        { flags: '--file <path>', description: 'Analyze specific file (default: auto-detect)' },
      ],
      async execute(ctx) {
        // 1. Find agent doc files
        const candidateFiles = ['CLAUDE.md', 'agents.md', 'AGENTS.md', '.claude/agents.md'];
        const foundDocs: string[] = [];
        for (const candidate of candidateFiles) {
          try {
            await fs.access(path.join(ctx.cwd, candidate));
            foundDocs.push(candidate);
          } catch { /* not found */ }
        }
        // If --file specified, use that instead
        if (ctx.args.file) foundDocs.splice(0, foundDocs.length, ctx.args.file as string);

        if (foundDocs.length === 0) {
          await ctx.ui.result({ success: false, title: 'Agents', summary: 'No agent instruction files found (CLAUDE.md, agents.md, AGENTS.md)' });
          return { success: false, summary: 'No agent docs found' };
        }

        // 2. Analyze each doc
        const reports: AgentDocMetrics[] = [];
        for (const doc of foundDocs) {
          const metrics = await analyzeAgentDoc(path.join(ctx.cwd, doc));
          metrics.efficiencyScore = computeEfficiencyScore(metrics);
          reports.push(metrics);
        }

        // 3. Generate suggestions (per D-18: suggest only, never modify)
        const allSuggestions = reports.flatMap(r => generateSuggestions(r));

        // 4. Compute overall score (average across all docs)
        const overallScore = Math.round(reports.reduce((sum, r) => sum + r.efficiencyScore, 0) / reports.length);

        // 5. Output
        const report: AgentDocReport = { docs: reports, overallScore, suggestions: allSuggestions };
        // Format and display...
        await ctx.state.set('agents.lastResult', { overallScore, docsAnalyzed: reports.length, timestamp: new Date().toISOString() });

        return { success: true, summary: `Agent doc efficiency: ${overallScore}/100 (${reports.length} file(s))`, data: report };
      },
    });
    ```

    Key: NEVER auto-generate or modify agent docs (D-18). Read-only analysis + suggestions.
