      7. **Dispatch agent**: `ctx.agent.run({ role: 'verification', prompt, permissions, timeout: 120_000 })`.
      8. **Parse result**: Extract JSON from agent output using regex for last JSON code block (same pattern as execute.skill.ts -- `const match = result.outputText.match(/```json\s*([\s\S]*?)\s*```/)`).       9. **Store in state**: `ctx.state.set('debug.lastResult', analysis)`.
      10. **Format output**: Display failure_type, root_cause, top 3 fix suggestions via `ctx.ui.result()`.
      11. Return `{ success: true, summary, data: analysis }`.
