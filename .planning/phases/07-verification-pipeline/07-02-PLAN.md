- `kind: 'prompt'` (uses agents in Layer 1 and 5)
- `description: 'Run 5-layer Swiss cheese verification pipeline'`
- Options: `--phase <number>` (which phase to verify), `--auto` (skip human gates for CI), `--strict` (fail on humanRequired findings)

Execute flow:
1. Resolve phase directory from --phase option or current phase from STATE.md
2. Read all PLAN.md files in phase directory, parse each with parsePlanMd()
3. Get git diff for the phase's work (use simple-git)
4. Execute all 5 layers SEQUENTIALLY (per D-01). Each layer gets the same ctx, diff, plans, phaseDir. Wrap each in try/catch -- a layer throwing should NOT stop subsequent layers. On catch, create a LayerResult with a single finding (severity: 'low', description: layer error message).
5. Aggregate all findings from all layers
6. Compute verdict: FAIL if any critical/high, WARN if any medium/low, PASS if empty
7. Check humanGateRequired: true if any finding has humanRequired=true
8. If humanGateRequired and NOT --auto: display findings summary, call ctx.ui.ask() for human approval. If --strict and humanGateRequired: verdict = FAIL.
9. Build VerifyReport with all layers, findings, verdict, timestamp
10. Write report to phase directory as VERIFICATION.md (formatted markdown with per-layer sections)
11. Store result in state: `ctx.state.set('verify.lastResult', { verdict, findingCount, timestamp })`
12. Return SkillResult with success = (verdict !== 'FAIL'), data = report
