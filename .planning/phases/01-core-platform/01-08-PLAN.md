    **packages/core/src/cli/lifecycle.ts:**
    Export function createLifecycle() that returns the full boot + execution + teardown sequence.

    Boot sequence (runs once at CLI start):
    1. loadConfig(cwd) -- get merged TOML config
    2. ensureSunDir(cwd) -- create .sun/ if needed
    3. createStateEngine(sunDir) -- init SQLite + file store
    4. scanSkillFiles(['packages/skills-*/src']) -- discover skill files
    5. resolveActiveSkills(discovered, config.skills) -- apply policy
    6. Create SkillRegistry, register active skills
    7. createAgentRouter({ providers, context, tracker }) -- init router with available providers
    8. createUiAdapter({ silent: config.ui.silent, json: config.ui.json }) -- select UI adapter
    9. Return { config, stateEngine, registry, agentRouter, uiAdapter, recommender }

    Execute hook (called per skill invocation):
    1. Create SkillContext via createSkillContext({ skillId, config, stateEngine, agentRouter, uiAdapter, recommender, registry, cwd })
    2. Run skill: registry.execute(skillId, context, options)
    3. After execution: get recommendations from recommender.getRecommendations({ lastSkillId, lastResult, ... })
    4. Display recommendations via ctx.ui.result() if recommendations exist
    5. Persist usage to state engine

    Teardown:
    1. stateEngine.close()

    Export createLifecycle as an async function that:
    - Returns { boot, createExecuteHook, teardown }
    - boot() returns the initialized services
    - createExecuteHook(services) returns the (skillId, options) => Promise<void> function
    - teardown(services) closes connections

    **packages/cli/src/cli.ts** (the actual entry point per D-03, CLI-01):
    ```typescript
    #!/usr/bin/env node
    import { createProgram, registerSkills, createLifecycle } from '@sunco/core';

    async function main() {
      const program = createProgram();
      const lifecycle = createLifecycle();

      try {
        const services = await lifecycle.boot(process.cwd());
        const executeHook = lifecycle.createExecuteHook(services);

        registerSkills(program, services.registry, executeHook);

        await program.parseAsync(process.argv);
      } catch (error) {
        // Render error via UI if available, otherwise console.error
        if (error instanceof Error) {
          console.error(`\n  Error: ${error.message}\n`);
        }
        process.exitCode = 1;
      } finally {
        // teardown handled by process exit
      }
    }

    main();
    ```
