Add a `headless` subcommand to the Commander.js program that:

1. Accepts: `sunco headless <skillCommand> [args...] [--timeout <ms>]`
2. Parses the skill command and remaining args
3. Forces SilentUiAdapter instead of InkUiAdapter
4. Runs the skill via the existing skill router
5. Captures the SkillResult
6. Prints JSON to stdout: `JSON.stringify({ success, summary, data, warnings, exitCode })`
7. Sets process.exitCode:
   - 0 if result.success === true
   - 2 if result.success === false AND result.data?.blocked === true
   - 1 otherwise
8. If --timeout provided: setTimeout(() => { process.exit(1); }, timeoutMs) at start

Implementation approach — add AFTER all skill registrations but BEFORE program.parse():

```typescript
// Headless mode subcommand
program
  .command('headless')
  .description('Run any skill in headless mode (CI/CD) — JSON output, structured exit codes')
  .argument('<command>', 'Skill command to run (e.g., status, auto, verify)')
  .argument('[args...]', 'Arguments to pass to the skill')
  .option('--timeout <ms>', 'Maximum execution time in milliseconds')
  .allowUnknownOption(true)
  .action(async (command: string, args: string[], options: { timeout?: string }) => {
    // Set timeout if provided
    if (options.timeout) {
      const ms = parseInt(options.timeout, 10);
      if (!isNaN(ms) && ms > 0) {
        setTimeout(() => {
          console.log(JSON.stringify({ success: false, summary: `Timeout after ${ms}ms`, exitCode: 1 }));
          process.exit(1);
        }, ms);
      }
    }

    // Find the skill by command name
    const skill = preloadedSkills.find(s => s.command === command);
    if (!skill) {
      console.log(JSON.stringify({ success: false, summary: `Unknown skill: ${command}`, exitCode: 1 }));
      process.exitCode = 1;
      return;
    }

    // Build a minimal SkillContext with SilentUiAdapter
    // Reuse the existing boot infrastructure but override the UI adapter
    try {
      // Import SilentUiAdapter
      const { SilentUiAdapter } = await import('@sunco/core');

      // Parse args into options object
      const parsedArgs: Record<string, unknown> = { _: args };
      for (let i = 0; i < args.length; i++) {
        if (args[i]?.startsWith('--')) {
          const key = args[i]!.slice(2);
          const next = args[i + 1];
          if (next && !next.startsWith('--')) {
            parsedArgs[key] = next;
            i++;
          } else {
            parsedArgs[key] = true;
          }
        }
      }

      // Create context with silent adapter (reuse existing context builder pattern)
      const ctx = await buildSkillContext({
        skill,
        args: parsedArgs,
        cwd: process.cwd(),
        uiAdapter: new SilentUiAdapter(),
      });

      const result = await skill.execute(ctx);

      // Determine exit code
      let exitCode = 0;
      if (!result.success) {
        exitCode = (result.data as Record<string, unknown>)?.blocked ? 2 : 1;
      }

      console.log(JSON.stringify({
        success: result.success,
        summary: result.summary,
        data: result.data,
        warnings: result.warnings,
        exitCode,
      }));

      process.exitCode = exitCode;
    } catch (err) {
      console.log(JSON.stringify({
        success: false,
        summary: err instanceof Error ? err.message : String(err),
        exitCode: 1,
      }));
      process.exitCode = 1;
    }
  });
```

Create `query.skill.ts` — a deterministic skill (kind: 'deterministic', zero LLM cost) that reads STATE.md + ROADMAP.md and returns a structured JSON snapshot:

```typescript
export default defineSkill({
  id: 'workflow.query',
  command: 'query',
  kind: 'deterministic',
  stage: 'stable',
  category: 'workflow',
  routing: 'routable',
  description: 'Instant JSON state snapshot — phase, progress, next action (no LLM)',

  async execute(ctx) {
    const statePath = join(ctx.cwd, '.planning', 'STATE.md');
    const roadmapPath = join(ctx.cwd, '.planning', 'ROADMAP.md');

    // Read state
    let phase: number | null = null;
    let status = 'unknown';
    let stoppedAt = '';
    try {
      const stateContent = await readFile(statePath, 'utf-8');
      const state = parseStateMd(stateContent);
      phase = state.phase;
      status = state.status ?? 'unknown';
      stoppedAt = state.stoppedAt ?? '';
    } catch { /* no state */ }

    // Read roadmap
    let totalPhases = 0;
    let completedPhases = 0;
    let nextPhase: string | null = null;
    try {
      const roadmapContent = await readFile(roadmapPath, 'utf-8');
      const roadmap = parseRoadmap(roadmapContent);
      totalPhases = roadmap.phases.length;
      completedPhases = roadmap.phases.filter(p => p.completed).length;
      const next = roadmap.phases.find(p => !p.completed);
      nextPhase = next ? `Phase ${next.number}: ${next.name}` : null;
    } catch { /* no roadmap */ }

    // Read cost from state
    let totalCostUsd = 0;
    try {
      const cost = await ctx.state.get<number>('usage.totalCostUsd');
      if (cost !== null && cost !== undefined) totalCostUsd = cost;
    } catch { /* no cost data */ }

    const snapshot = {
      phase,
      status,
      stoppedAt,
      progress: { total: totalPhases, completed: completedPhases, percent: totalPhases > 0 ? Math.round((completedPhases / totalPhases) * 100) : 0 },
      nextAction: nextPhase,
      costs: { totalUsd: totalCostUsd },
      timestamp: new Date().toISOString(),
    };

    const summary = `Phase ${phase ?? '?'}, ${completedPhases}/${totalPhases} complete, ${status}`;

    await ctx.ui.result({ success: true, title: 'Query', summary });

    return { success: true, summary, data: snapshot };
  },
});
```
